How to beep with system speaker.


Keep the SCP.EXE and BEEPER.SYS on the same directory and then run the SCP.EXE.
  Within a microsecond, you would hear the PC broad beeps "Bi - - - ".


Tested on: Windows NT4.0+sp6, 2000, XP & Server 2003
